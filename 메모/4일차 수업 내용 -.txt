<포토샵>
영역 확인 방법
- 선택 툴 클릭 후 눈금따라 그어서 확인하기
- window ->info -> 들어가서 영역값 확인

포토샵 이미지를 코딩으로 분할해 가는 법
- 완성된 포토샵 중 이미지와 연결해있는 레이어드 중복 클릭
- 우클릭 후 duplicate layer 클릭 후 new클릭
- 상단바 image 클릭 후 trim 확인
- 이미지로 저장
- html -> img파일로 저장

<코딩>
-> 포토샵으로 만든 사진을 가져와서 코딩사용해서 만들어보기
-> 코팅에서는 글자 위아래 여백 크기는 없앨 순 없음
    = 그래서 margin 값을 포토샵 설정 크기보다 줄여서 작업
