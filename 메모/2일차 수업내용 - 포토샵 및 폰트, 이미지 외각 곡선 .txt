2. 코팅
div에게 각 이름을 붙이고 싶으면 class="(이름)"으로 작성
 - class="(*)"
(*)여기에 들어가는 이름은 갯수 상관없이 지어줄 수 있음 but 띄어쓰기 중요
ex) <div class="p p1">padding1</div>

* 모서리 곡선 만들기
radius = 곡선
bordround_radius: **px;
**마우스 올렷을 시 색상 변화 만들기
*r1:hover{} 후 {]안에 색상 넣으면 마우스 올렸을 시 색상 바뀜

* 폰트 바꾸기
body에 제목(h1)과 부제목(p) 넣기
해당하는 폰트 가져와서 코드 넣고 style 밑에 잇는 * 위에 코드 붙여넣기
맨 처음 에 있는 font-family 가져와서 해당 코드 {}안에 넣기
